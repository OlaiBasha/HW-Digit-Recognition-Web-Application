# شرح جزئية Machine Learning

## الفكرة العامة

المشروع يحتاج أن يرفع المستخدم صورة لرقم مكتوب بخط اليد، ثم يرجع النظام الرقم المتوقع. لذلك قسم الـML مقسوم إلى جزأين:

1. تدريب النموذج على صور أرقام مكتوبة بخط اليد.
2. تشغيل API يستقبل الصورة من React ويرجع النتيجة.

## التدريب

استخدمنا Dataset اسمها MNIST. هذه الداتا تحتوي صور أرقام من 0 إلى 9، وكل صورة حجمها 28x28 وبالأبيض والأسود.

الملف المسؤول عن التدريب هو:

```text
train_model.py
```

هذا الملف يعمل الخطوات التالية:

- يحمل MNIST.
- يحول قيم البكسلات من 0-255 إلى 0-1 حتى يكون التدريب أسهل.
- يبني CNN model مناسب للصور.
- يدرب النموذج على بيانات التدريب.
- يختبر النموذج على بيانات الاختبار.
- يحفظ النموذج داخل:

```text
model/mnist_cnn.keras
```

## شكل النموذج

النموذج المستخدم هو Convolutional Neural Network لأن هذا النوع ممتاز مع الصور. يحتوي على:

- Conv2D لاستخراج مميزات من الصورة.
- MaxPooling2D لتقليل الحجم مع الحفاظ على المعلومات المهمة.
- Flatten لتحويل الصورة إلى vector.
- Dense لاتخاذ القرار.
- Dropout لتقليل overfitting.
- Softmax لإرجاع احتمال كل رقم من 0 إلى 9.

## API

الملف المسؤول عن تشغيل السيرفر هو:

```text
api.py
```

هذا الملف يشغل FastAPI وفيه endpoint:

```text
POST /predict
```

الفرونت يرسل الصورة باسم `file`، والباك إند يعمل preprocessing للصورة:

- يحولها إلى grayscale.
- يعكس الألوان إذا كانت الصورة رقم أسود على خلفية بيضاء.
- يقص الفراغ حول الرقم.
- يضع الرقم في منتصف صورة 28x28.
- يطبع القيم بين 0 و1.

بعدها يرسل الصورة للنموذج ويرجع:

- الرقم المتوقع.
- نسبة الثقة.
- احتمالات كل الأرقام.

## أوامر التشغيل

```bash
pip install -r requirements.txt
python train_model.py
uvicorn api:app --reload --host 0.0.0.0 --port 8000
```

## ربط React

React يرسل الصورة إلى:

```text
http://localhost:8000/predict
```

باستخدام `FormData` وحقل اسمه `file`.

مثال جاهز موجود في:

```text
react_predict_example.jsx
```
